# 🤖 AI 工作助手 — 智能办公工具箱

> 输入要点，AI帮你生成专业内容。周报、会议纪要、简历优化，一站式搞定。

**🌐 在线体验：[weekly-report-ai-two.vercel.app](https://weekly-report-ai-two.vercel.app)**

---

## 🧰 已上线工具

### 📝 AI 周报生成器
输入本周工作要点，AI自动生成结构清晰的专业周报。
- 4种风格：专业汇报 / 成果导向 / 极简高效 / 技术风
- 30秒出稿，一键复制

### 🎙️ AI 会议纪要
粘贴会议记录，自动整理成标准化纪要格式。
- 自动提取：议题、讨论要点、决议、待办事项
- 支持语音转文字后的凌乱文本

### 📄 AI 简历优化器
大厂PM视角 + STAR法则，让简历脱颖而出。
- 重写每条经历，突出量化成果
- 专业语言润色
